# 拍卖平台章节实验结果分析仓库

该仓库主要是用来分析拍卖平台设计章节的实验产出的结果日志，然后得到实验统计数据和图表

说明：

1. 在运行完实验后，服务端和模拟客户端程序都会产出日志文件，可以见具体的仓库实现。

2. 所以首先将产出的日志文件拷贝至当前目录，运行`copy_logs.py`文件即可，记得修改保存的位置，不同的实验请保存到不同的文件夹中

3. `bsc_rpc`包下是分析本文的日志的入口，分别有客户端和服务端日志分析程序

4. `share_node`包下是分析`MEV-Share Node`的日志的入口，同样有客户端和服务端日志分析程序

5. 为了得到分析结果，修改上述两个包下的代码中带分析的日志的路径，然后运行对应的`py`文件即可，最终会在控制台输出分析结果

6. 对于文章中的结果图，他的生成程序在image包下，指定日志文件路径运行，即可生成对应的结果图

注意：python环境为3.9。
